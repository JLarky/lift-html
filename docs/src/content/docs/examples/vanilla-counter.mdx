## Explanation

This example is there to contrast the implementation with the version using lift-html. Initial component markup is generated by the server, then once client side implementation of custom element is loaded then `disabled` attribute is removed and click handler is added to the button. See also versions using [@lift-html/solid](/examples/solid-counter/) and [@lift-html/core](/examples/core-counter/).
